This is a starter Unity project with packages installed for GD2S03 Assignment 3.

It has been built with Unity 2022.3.50f1

It includes packages for Google Admob, Google Play Games Services and Unity IAP.

This has been provided to help students get thier project running and avoid gradle errors. 

Get Started:
- Set Project to Android (Build Settings)
- Set Company and Product Name (Player Settings)
- Create a Custom Keystore 

Monetisation
- Set up scripts to initialise and display test ads within the app.
- Create and connect a cloud project for IAP

Play Games Services
- Upload a signed build to play console to unlock Play Games Services
